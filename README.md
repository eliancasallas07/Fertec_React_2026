
# Catálogo de Cerraduras

Este es un proyecto web sencillo hecho con React y JavaScript para mostrar un catálogo de cerraduras.

## ¿Cómo iniciar?

1. Instala las dependencias si es necesario:
   
	npm install

2. Inicia el servidor de desarrollo:
   
	npm start

El sitio estará disponible en http://localhost:3000

## Estructura
- Catálogo básico de cerraduras (puedes modificar o ampliar los productos en `src/App.js`).

---

Este proyecto es solo una base, puedes personalizarlo según tus necesidades.
